9-1 HTML Inline CSS

Resources

  • HTML CSS

Lab Exercise

  1. Open VS Code and create a new document named 9_1_inline.html and save it to your HTML folder.

  2. Give the document the title: 9_1_inline.

  3. Add the following html to your document.


    <!DOCTYPE html>
<html> 
<head> 
</head> 
<body> 
<h1>Pool Rules</h1> 

<p>Lifeguard not on duty. Swim at your own risk.</p> 
<p>Children under the age of 16 must be accompanied by an adult.</p> 
<p>No Diving.</p>

</body> 
</html>

  4. Make the following modifications using inline styling:

    • Change the body element's background color to lightyellow.

    • Change the h1 element's text color to darkred.

    • Change the h1 element's font-family to Verdana.

    • Change each of the three paragraph element's text color to blue.
