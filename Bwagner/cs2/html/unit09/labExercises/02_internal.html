9-2 HTML Internal CSS

Resources

  • HTML CSS

Lab Exercise

  1. Open VS Code and create a new document named 9_2_internal.html and save it to your HTML folder.

  2. Give the document the title: 9_2_internal.

  3. Add the following html to your document.


    <!DOCTYPE html>
<html>
<head>
</head>
<body>
<h1>Pool Rules</h1>

<p>Lifeguard not on duty. Swim at your own risk.</p>
<p>Children under the age of 16 must be accompanied by an adult.</p>
<p>No Diving.</p>

</body>
</html>

  4. Make the following modifications using internal styling:

    • Change the body element's background color to whitesmoke.

    • Change the heading element's text color to steelblue.

    • Change the heading element's font to veranda.

    • Change the paragraph element's text color to lightcoral.

    • Increase the paragraph element's font size to 150%.
