6.9 Palindrome

Description

Use a stack and a queue to implement a palindrome checker.

Pseudocode

Read a string
Traverse the string
   - push each character on to a stack.
   - enqueue each character into a queue.
When you reach the end of the line, pop the stack and dequeue the queue until you can determine whether or not the input string is a palindrome.

Sample Run

Enter a string --> racecar

This is a palindrome.

Try Again(Y/N)?  Y

Enter a string --> noon

This is a palindrome.

Try Again(Y/N)? Y

Enter a string --> dog

This is not a palindrome.

Try Again(Y/N)? N
