ChangeCase

Directions

Write a program that allows a user to enter an unknown number of words. Input should stop when the user types the sentinel value "stop". When the input process is complete, the program should display each word in its original form, in all upper case letters, and in all lower case letters.

Since you do not know how many words the user will enter you will need to make the array's physical size large enough to accommodate a reasonable amount of words. You will also need to include a variable that keeps track of the array's logical size, that is, the actually number of words that have been entered into the array.

The String class includes two methods that return strings that have been converted into all uppercase and lowercase letters. As an example, the following code

String str = "Computer";
System.out.println(str.toUpperCase());
System.out.println(str.toLowerCase());

produce this output

COMPUTER
computer

Use printf to display the output in columnar format.
