LetterGrade

Directions

Write a program that reads a numerical grade from the user then prints a corresponding letter grade according to the scale below.

Numerical Grade Letter Grade
100 - 97 A+
96-94 A
93-90 A-
89-87 B+
86-84 B
83-80 B-
79-77 C+
76-74 C
73-70 C-
below 70 F

The use of methods is optional.
