Directions

In this assignment you are going to complete three methods: build1, build2, and build3.

  1. Complete method build1. In method build1 create 5 nodes and store them in the instance variables provided at the top. Assign a value of 1 to node1, a value of 2 to node2, and so on. Using only a constructor link the 5 nodes together to form a linked list. You will need to instaniate (create) the nodes in reverse order starting with node 5. Link node 1 to node 2, node 2 to node 3, and so on.

  2. Complete method build2. Create a linked list again using the same node variables, but this time use a constructor and the setNext method to build the list. First create the 5 nodes using the ListNode constructor setting each nodes next variable to null. Then using the setNext method link the nodes together from front to back. Link node1 to node2, node2 to node3, and so on.

  3. Complete method build3. Create a linked list a third time using the variables head, newNode, and cur. Create the list with 20 nodes number from 1 to 20. Use a for loop in your implementation.
